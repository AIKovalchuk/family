В реализации приложения используется микросервисная архитектура, в качестве тестового контура выбран сервис Heroku, требующий отдельный репозиторий для каждого сервиса, по этой причине каждый микросервис лежит в своем репозитории.

Ссылки:
	https://github.com/boriskezikov/family-gateway - API-GATEWAY
	https://github.com/boriskezikov/family-notification - Notification service
	https://github.com/boriskezikov/family-auth - Authorization service
	https://github.com/boriskezikov/family-exemption - Analysis service
